# Hardware-Y-redes-Juan-Torres
"Plataforma educativa interactiva sobre componentes de hardware de computadoras, diseñada para estudiantes de secundaria (grado 10-11). Integra modelos 3D navegables de RAM, CPU y placa madre, contenido pedagógico estructurado y un asistente virtual con inteligencia artificial que responde preguntas en tiempo real sobre hardware y redes.
